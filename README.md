# Event+ 

<img src='https://github.com/gabrielvictor0/imagens_readme/blob/main/event-plus.png' />

<p> O Event+ é um sistema de gerenciamento de eventos da escola Senai Paulo Antonio Skaf. Ele foi desenvolvido com o intuito de facilitar a divulgação dos eventos no ambiente escolar e para melhorar a interação dos alunos com os eventos. </p>
